  • Project 3: 多种数据的自杀风险预测分析

    This project analyzes global suicide rates using WHO and UNDP data. Key factors such as age, gender, and education influence suicide rates, with older individuals and males being most at risk. Machine learning models, like Random Forest, were used to predict suicide rates, achieving 81.59% accuracy. The findings highlight the need for targeted mental health interventions in high-risk regions.

  • Project 5: 玩家行为数据分析与胜率预测建模 (Machine Learning)

    This project aimed to predict the outcome of League of Legends matches using machine learning models. The data was sourced from Riot API and includes the first 15 minutes of gameplay from over 24,000 solo queue matches on European servers. The dataset features 28 variables, including kills, gold, and objectives like dragon kills, with the target variable being whether the blue team wins.

    • 基于 Riot API 抓取的英雄联盟玩家历史对局数据，提取击杀、助攻、经济等多维行为特征

    • 构建逻辑回归、随机森林、SVM 等模型预测个人胜率，最高准确率达 75%

    • 通过特征重要性分析揭示影响胜率的关键操作因素，训练建模思维与解释能力。
